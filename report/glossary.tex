
list of terms and definitions:
\begin{itemize}
    \item \textbf{Process} - An instance of a program that is being executed. It contains the program code and its current activity.
    \item \textbf{Pipeline} - A series of data processing stages where the output of one stage is the input to the next.
    \item \textbf{Thread} - An isolated unit of execution that shares memory space with other threads of the same process.
    \item \textbf{Job} - A process or a pipeline that is started or scheduled to run by the user in a shell session or from the terminal.
\end{itemize}
